# BRIXXSPACE - Premium Construction & Infrastructure

South Tamil Nadu's Premier Construction & Infrastructure Company. Building excellence with trust for over 30 years.

## Overview
Brixx Space connects clients with top-tier construction consultation, project advisory, and architectural planning.

## Features
- **Client Portal**: Project tracking and document management.
- **Service Listings**: Detailed breakdown of our expertise.
- **Portfolio**: Showcase of over 500+ completed projects.
- **Contact**: Direct consultation booking.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Development
To run the project locally:
```bash
npm install
npm run dev
```
